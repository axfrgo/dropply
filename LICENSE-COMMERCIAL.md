# Commercial / Proprietary Components

The open-source license in [LICENSE](./LICENSE) applies to the public desktop core in this repository.

The following are outside the open-source grant unless explicitly provided under a separate written agreement:

- hosted Dropply cloud services
- managed relay infrastructure
- account systems
- billing systems
- private/team workspace services
- internal admin tooling
- proprietary enterprise integrations
- private deployment automation and operations code

Any future proprietary modules distributed separately should include their own commercial terms and support policy.
